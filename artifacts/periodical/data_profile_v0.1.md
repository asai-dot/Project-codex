## 4. 注記・手法

- tsuukan(通巻)系列誌: 197誌 / 巻号(vNnM)系列誌: 162誌。
- isbn_per_issue誌: 585誌(別冊ジュリスト等、識別子=NCID#suffix形式)。ISBN重複group: 0。
- 飛び番号は**両端present・run長<=5**の孤立欠落のみ採用(データ未収録の時代帯を誤検出しない高精度設定)。
- 評釈判定: タイトル内の(元号+判/決/審/裁決)括弧 もしくは 判例研究|判例評釈|判批 等のキーワード。
- 連載: article_series_v0.1.csv に article_id が含まれる記事。特集: issue_feature_v0.1.csv の issue_id 一致。
- read-only: 入力CSVのみ参照。canonical/DB/edgeへの書込なし。
