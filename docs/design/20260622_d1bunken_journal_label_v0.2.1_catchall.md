## 背景（v0.2 の残課題）

v0.2 実行後、`法学` が canonical 最大の **46,279件**。中身を `掲載誌等` 先頭で割ると
別誌の寄せ集めと判明（＝広すぎる検索語「法学」で D1 を全部入りで落としたフォルダ）:

| 隠れていた実誌 | 件数 |
|---|---|
| 法学セミナー | 8,535 |
| 月刊法学教室 | 5,959 |
| 法学研究（慶応） | 1,833 |
| **法学協会雑誌（評釈4位）** | **1,297** |
| 立命館法学 / 法学新報（中央）/ 阪大法学 … | 各〜1,000 |

→ データは在るが誌名が「法学」に潰れ、**評釈4位の法学協会雑誌が独立カウントされず埋もれていた**。
再取得は不要。**ラベラー側で `掲載誌等` を使い割り直す**のが正解。

## 方針

- 各 canonical フォルダで「`掲載誌等` 先頭 == フォルダ名」の一致率(agreement)を測る。
- `件数≥300 かつ agreement<0.5` を **catch-all** と判定し、そのフォルダのレコードだけ
  `掲載誌等` 先頭で誌名を割り直す。純粋フォルダ（ジュリスト等, agreement高）は **v0.2 のまま不変**。
- catch-all で救出した誌は、独立フォルダで再取得済みの同名誌と canonical 一致で **自動統合**。
- 非破壊・冪等: `<build>/labeled_v0.2.1/` に新規出力、件数(282,761)不変。

`掲載誌等` 先頭抽出: NFKC → 先頭の括弧/引用記号除去 → 最初の数字(巻号)で切る →
大学名などの括弧修飾は残す（誌の区別に効く）。
