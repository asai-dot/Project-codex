## 0. 形式の決定（司令塔判断）

最終成果物は xlsx 指定だが、**stdlib での装飾付き xlsx 再生成は非効率かつ git 追跡に不向き**、かつ
実行環境に openpyxl が無い（`python3 -c "import openpyxl"` で `ModuleNotFoundError`）ため、
**CSV(1シート=1ファイル) + 本インデックスMD を canonical 形式**とする。

- 各シート: `ledger_v0.2/NN_<sheet>.csv`（UTF-8, ヘッダ付き, `csv.reader` パース検証済）。
- xlsx 組立は **openpyxl 導入後の別工程**（捏造・空 xlsx は作らない）。
- front-matter deliverables の `..._v0.2.xlsx` は **本MD + ledger_v0.2/*.csv 群で代替**（上記理由）。
