# データ分析レポート v0.1 — 何がデータから言えるか

## エグゼクティブ・サマリー（3つの戦略所見）
1. **成長エンジンが「低転換チャネル」に偏った**：2024年以降 公的機関(法テラス系)が最大流入(0→33→30→20/年)。しかし転換率は **公的機関24% vs 紹介51%**。高価値の紹介は年12-17件で横ばい。→ 量は増えたが、最も決まる紹介チャネルが未開拓。
2. **案件が1名の弁護士に極端集中**：受任案件 弁護士A=747件 / 弁護士B=26件（Aが約97%）。相談転換も A38.6% vs B21.4%。→ キーパーソン・リスクと処理能力上限。
3. **運用は几帳面、経営分析は盲目**：担当・期限・日付は高充足。一方 成果94%空白・失注理由81%空白・流入経路86%空白・待ち先70%空白・分野98%空白。→ 勝率・失注要因・チャネルROIが測れない。**統合の最大価値はこの空白を観測データで埋めること**。

## データ品質の構造（最重要パターン）
- **充足（運用系）**：担当弁護士98% / 期限リンク100% / 終結日91% / 係属案件の放置ほぼ無し。
- **空白（分析系）**：成果94% / 相談未到達理由97% / 分野98% / 流入経路86% / 失注理由81% / 待ち先70% / 受任確度65%。
→ 「誰が・いつ・何の期限で」は分かるが「どこから来て・なぜ決まり/負け・何の分野で・どう終わったか」が分からない。

## 推奨アクション（優先順）
1. **流入経路・失注理由の補完**：最大の分析ROI。Gmail/Boxの自然発生データから推定→人手承認(G-19)。源アラート(継続44件)は実装済。
2. **紹介チャネルの開発**：最も決まる(51%)が伸びていない。紹介元は分散(単一の大口なし)＝関係構築を面で。
3. **担当集中の緩和とデータ駆動トリアージ**：弁護士Aの時間を高転換リードに集中させる振り分け。
4. **成果(勝敗)トラッキングの確立**：94%空白では勝率が見えない。法律事務所として致命的盲点。
5. **収益分析は会計オブジェクト取込で**：Earningsは使えない。着手金/報酬/実費/タイムチャージ + MoneyForward/通帳が必要。
