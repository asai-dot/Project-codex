# 至誠堂「標準本」シグナル → 蔵書の重み付け候補 (2026-06-05)

> 目的: 弁護士の文献評価を待たず、**外部の権威ある「標準本」推薦**を取り込み、蔵書(asai-bookshelf)に
> 「標準本フラグ（重み）」を立てる候補を作る。Stage2 gold（事件類型×重み付き推薦本）の種。
> **DBには書いていない**（フラグ列/側テーブルの実装はスキーマ所有の codex/花岡 と合意後）。データファイルのみ。

## ソース（至誠堂書店オンラインショップ・公開カタログ）
- **新人弁護士がよく買う本**: 企業法務系(tid=5, 103冊) / 個人法務系(tid=6, 142冊) — 最も選別された推薦。
- **分野別スタンダード書籍**: 30分野(cid=4〜33)。各分野の定番。
- 取得: ブラウザ描画→AJAX `/aj/tags?tid=` ・ `/aj/items?cid=`（`category_id` でなく `cid`）。書名・著者・出版社・ISBN を抽出。
- **外部標準本ユニーク 1,036冊（全てISBN付き）**。tier: 両方(B)188 / 分野のみ(F)818 / 新人のみ(N)30。

## 突合（read-only・ISBN13正規化）
- 我々の `biblio.bib_records` は ISBN付き 5,403件（全13桁）。蔵書 source=asai-bookshelf と突合。
- **1,036冊中 474冊が実棚(asai-bookshelf)に存在** ＝ フラグ候補。
  - **両方(新人＋分野標準)=81冊（最重要・最も定評）**
  - 新人リスト掲載 計101冊 / 分野標準 計454冊。
- bencom-library 側のマッチは0（ISBN欠落 or 重複なし。実棚評価には asai-bookshelf を使う）。

## 分野別カバレッジ（実棚保有 / 至誠堂標準数）
強い: 商法・会社法 66/108、民事手続 44/74、労働 36/67、破産・再生 31/50、民法全般 29/56、契約書 17/21。
**薄い（主力なのに手薄＝A1の「届かない領域」と符合）**: 交通事故 12/35、行政 7/35、外国法 7/32、消費者三法 5/19、税法 4/13、医療 4/11、金融商品取引 1/19、国際私法 0/5、債務整理 0/4（※債務整理の実体は破産・再生51/31でカバー）。
→ 「橋を張る価値が高い領域（A1の raw_probe_hit_but_no_term_hit）」と「標準本を持っている領域」が重なる所＝**最初に bib_terms を張る最有力候補**。

## 成果物（`build/search_bench/`）
- `shiseido_standard_books_master.csv` — 外部標準本 1,036（isbn13/書名/出版社/tier/分野/新人リスト）。重みの原典。
- `shiseido_shelf_standard_flag_candidates.csv` — 実棚の標準本 474（bib_id付き）。**これが蔵書フラグ候補**。
- `shiseido_match_summary.json` — 集計（tier別・分野別カバレッジ）。

## 重み付けの考え方（Stage2 で使う）
- `standard_tier`: both(新人＋分野) > newcomer_only > field_only の順で重み。
- `fields`: 事件類型→本の対応（A1の事件類型と接続）。
- これは「①事件類型→本の関連度」「②本の良し悪し」の**外部シードの一層**。最終的な gold は、弁護士の文献評価で
  この層の上に事務所固有の重みを足して確定する（人手はそこだけ）。
