# 多ソース標準本 重ね合わせ（endorsementカウント）2026-06-05

> 9系統の権威ソースの推薦書を横断し、「**何系統が推すか**」を蔵書(asai-bookshelf 6,524)に重ねた。
> 多系統一致＝偏りが打ち消された“本物の定番（硬い本）”。Stage2 gold の重みの外部シード。**DB非書込**（フラグ列はcodex/花岡合意後）。

## データと方法
- ソース: #1至誠堂(1036/ISBN) #2BL(40) #3勁草MAP全8回(~95) #4契約ウォッチ(55) #5businessandlaw(6) #6弁ドットコム(Vol.72=6/**約70号は深掘り未**) #7LEGAL JOB BOARD(15) #8大阪弁護士会(24) #9二弁(28)。
- 媒体・弁護士会系はISBN記載なし→**書名 work-key 正規化**（NFKC＋版/括弧/記号除去）で蔵書・至誠堂と照合。
- 集計: `build/search_bench/sources/source_books.tsv`（統合書名）→ `match_multisource.py`。

## 結果
- 統合した独立ソース work数 = **1,171**、うち**748 が実棚に存在**。
- 蔵書側で1系統以上から推された本 = **903冊**。
  - endorsement分布: **5系統=10冊 / 4系統=5冊 / 3系統=44冊 / 2系統=164冊 / 1系統=680冊**。
- **クロスソース核（≥2系統）= 80 work、うち実棚 63**。

## 留保（重要・過信しない）
- **書名一致は近似**: 媒体系にISBNが無いため work-key（版・括弧除去）で照合。「労働法」「会社法」等の短い総称は**著者違い・版違いを1 workに畳む**（菅野/水町/荒木の労働法が同一keyに集約）。系統数は「その定番書群を何系統が推すか」の近似であり、版・著者の厳密同定ではない。
- 蔵書側titleのみで照合（著者列は未取得）＝短title総称で過剰結合の可能性。endorsement_count は**目安**。
- ISBNでの厳密突合は至誠堂(474)のみ確定。媒体系の厳密化は書名→ISBN解決を別途。
- #6 弁ドットコムタイムズ 約70号バックナンバー、#3勁草ls_map10(まとめ)は未取得（深掘り対象・no silent truncation）。
