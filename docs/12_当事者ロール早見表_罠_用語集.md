# 当事者ロール早見表 ＋ 踏みやすい罠 ＋ 用語集（AIフィット オンボーディング1式）

- 作成日: 2026-06-07
- 用途: ALOコネクト（AIフィットが構築）が、メール/電話/FAX等を**正しい当事者構造でSF案件に紐付ける**ための、法律実務ドメインの前提知識。一般AIベンダーには非自明な点を1枚に。
- 守秘: 匿名化版（AIフィット引き渡し可）。

---

## 1. 当事者ロール早見表（誰からの連絡が、どの案件のどのロールか）

| ロール | 説明 | 紐付けの扱い | SF対応 |
|---|---|---|---|
| **依頼者** | 案件の当事者本人（個人/法人） | matter の中心 | `party`(client) / Account・Contact |
| **相手方** | 紛争・取引の相手 | **当該matterに紐付く**（別案件ではない） | `OpponentParty__c`(126) |
| **相手方代理人** | 相手方の弁護士・士業 | **自所の当該matterの活動**として紐付く。差出人が他事務所でも非案件ではない | matter_party(role=counterparty_counsel) |
| **関係者** | 証人・親族・連帯保証人・担当者 等 | matterに紐付くが当事者ではない | `RelatedContact__c`(1,744) |
| **裁判所** | 送達元・期日連絡 | 当事者でなく court。**事件番号が最強キー** | `JurisdictionCourt__c`(160) |
| **紹介元** | 法テラス/弁護士会/遺言相続センター/夜間相談 等 | **入口（intake）であって当事者ではない**。分野推定に使う | （ラベル/メタ） |
| **協業先** | 共同のFA・会計士・司法書士・他事務所弁護士 | 当事者でなく協業者。matterには関与ロールで紐付く | matter_party(role=collaborator) |
| **自所（手続主体）** | ALO弁護士が**破産管財人/後見人/保佐人/遺言執行者/監督委員** | **ALO＝手続の主体**で依頼者ではない。通常と当事者構造が逆 | matter(種別=管財/後見等) |
| **共同受任・特約** | ALO内で複数弁護士が担当（例: 主担当＋特約） | 1matterにN担当 | matter_party(role=internal_attorney) |
| **顧問先** | 顧問契約のある継続クライアント | 「土壌」。個別依頼が立てば別matter | `AdvisorContract__c`(25) |
| **保険（LAC等）** | 弁護士費用保険（交通事故等） | 請求先＝保険会社。案件は依頼者のもの | （会計/請求） |

**最重要の非自明ポイント（一般ベンダーが誤る）**
1. **相手方・相手方代理人からのメールも「自分の案件」に紐付く。**
2. **自所が管財人/後見人/遺言執行者のときは、ALOが当事者（依頼者不在に見える）。**
3. **紹介元・協業先は当事者ではない。** 同姓異人にも注意。

## 2. 踏みやすい罠 1枚（負の知識）

| よくある誤り | 正しい扱い |
|---|---|
| @メンション = 担当 | 利益相反確認の@は**非担当**（消去法）。`assignee_prior` 止まり、単独確定しない |
| 同姓 = 同一人 | 別人は別party。住所/生年/事件番号/メール一致でのみ併合 |
| メール件数が多い = 重要案件 | 研修・対外活動は件数多くても**非案件** |
| 紹介元 = 当事者 | 入口に過ぎない |
| 顧問先の定例連絡 = 個別案件 | 顧問は「土壌」。個別受任と分離 |
| 旧社名/婚氏で別物扱い | 旧社名・旧姓は**別名**（名寄せ。ただし自動確定しない） |
| 中黒/全半角を潰せば安全 | 潰しすぎると**別法人を誤併合**。正規化キーは候補用、確定キーにしない |
| 相手方代理人＝別組織だから非案件 | **自所の当該案件**の活動 |
| 自所名が差出/宛先＝内部メール | 管財/後見/遺言執行では**自所が手続主体**。除外しない |
| SFに無い＝該当案件なし | **SF未起票が約30%**（Pattern E）。Box実体で起票候補に |
| Boxに背骨IDが無い | **ある**（`_alo_dir.json` の `sf`／`alo_dir_index_accum.json`） |
| 3月資料のSFオブジェクトを実在前提 | **ghost/遺物あり**（`leala__Procedure__c`不存在・`Matter__c`遺物）。実describe優先 |

## 3. 用語集（業務用語 → データ上の現れ方）

| 用語 | 意味 | データ上の現れ方 |
|---|---|---|
| 受任案件 | 正式に受任した案件 | `leala__Business__c`(779) / 接頭辞 a0A / Box「1受任案件」 |
| 相談 | 受任前の相談 | `leala__Consultation__c`(230) / a0D / Box「2相談」 |
| 顧問契約 | 継続的なお付き合いの契約 | `leala__AdvisorContract__c`(25) / a04 |
| 顧問相談 | 顧問先からの相談（関連交渉の格納先） | `leala__AdvisorConsultation__c`(26) / a03 |
| 裁判外業務 | 訴訟によらない業務（交渉・契約等） | Box「3裁判外業務」 |
| 管財（破産管財人） | 破産手続で財産を管理・換価する役 | 自所＝手続主体。matter種別 |
| 後見/保佐/補助 | 判断能力不十分な人の財産管理・支援 | 自所＝後見人等。matter種別 |
| 遺言執行者 | 遺言の内容を実現する役 | 自所＝執行者 |
| 民事再生/事業再生 | 会社を立て直す手続 | M&A/再生系matter |
| 経営者保証GL | 経営者保証ガイドライン（保証債務整理） | 個人保証の整理案件 |
| M&A | 事業譲渡・株式譲渡等 | 受任/顧問相談。協業FA・会計士・司法書士が関与 |
| LAC | 弁護士費用保険（交通事故等） | 請求先＝保険会社 |
| 紛議調停 | 弁護士会の紛議調停 | 紹介元/手続 |
| 法テラス | 日本司法支援センター（民事法律扶助） | 紹介元・費用区分 |
| 事件番号 | 「令和○年(ワ)第○号」等 | `court_case_number`（同名別事件の最強キー） |
| 預り金/実費 | 案件ごとの預り金・実費 | `Expense__c` / 会計 |
| 送達 | 裁判所からの書面交付 | 郵便(`PostalMatter__c` 7,098) |

## 4. AIフィットへの一言
- **紐付けは「当事者の役割構造」を踏まえて初めて正しくなる**（相手方・自所主体・協業先・紹介元の区別）。
- 詳細な実例は [`11_難所カタログ`](11_難所カタログ_ゴールドセット_v0.1.md)＋`data/goldset_anonymized.csv`。
- Box→SFの確定対応は `_alo_dir.json` / `alo_dir_index_accum.json` を使う（[`10`](10_AIフィット向け前提資料_引き渡しチェックリスト.md)）。
