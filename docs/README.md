## 3行サマリ

1. **Gmail**: 1,181案件ラベルが Salesforce ID 付きで紐付け済（受任案件780＋相談230で約85%）。
2. **Salesforce / Box**: オブジェクト体系とフォルダ分類が一致し、`sf_record_id` を背骨に依頼者名・事件名で機械照合できる状態。
3. **判断ロジック**: 7段階の優先順位＋当所固有ルール（旧社名・複数案件分岐・相談⇄受任・非案件の研修等）を言語化。機械が迷う場面だけ人に確認を促す設計。
