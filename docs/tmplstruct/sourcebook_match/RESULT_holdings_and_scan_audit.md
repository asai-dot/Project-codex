## B. 自炊フォルダ 4点セット監査（実地確認）

**フォルダ**: `ALO共有フォルダ / し＿自炊書籍データ`（Box folder_id 350436318926）
**正準ルール**(`スキャンルール.txt`): 600dpiで `01_全体` / `02_目次` / `03_索引or用語集` / `04_奥付` の4ファイル。999頁超は分割（01_全体_書名1/2）。

| 指標 | 値 |
|---|---:|
| 書籍フォルダ数 | **1,148**（≒1,145冊＋ルール文書等） |
| `01_全体` PDF | 1,427（分割本で水増し） |
| `02_目次` PDF | 1,161 |
| `03_索引` PDF | 1,157 |
| `04_奥付` PDF | 1,170 |

→ 目次/索引/奥付がいずれも **書籍数(1,148)とほぼ同数(1,157〜1,170)** ＝ **大多数が4点セット完備**。
索引(1,157)が最少で、辞書系など索引が無い本もあるため、**索引欠けが完全性の主なギャップ**。

### 未完了の明示フラグ `（未）`
登録/確定が保留の本は `（未）` 接頭辞で管理されている（例: `（未）三訂版企業労働法実務入門`・`（未）情状弁護アドバンス書式データ`・`（未）新・会社法実務問題シリーズ定款・各種規則の作成実務第4版`・`（未）民事保全非訟マニュアル`・`（未）民事再生実践マニュアル第2版`・`（未）詳解労働法第4版`）。
**実地確認**: `（未）`はファイル欠落ではなく「システム未登録/未確定」を意味し、4ファイル自体は揃っている例が多い（情状弁護アドバンス・三訂版企業労働法とも 01〜04 揃いを確認）。

### 出典本×自炊の実地検証（4点セット確認済）
