  **DB search_path 末尾に lawtime を追記**（`"$user", public, lawtime`。public 優先維持・lawtime は fallback）。

## 再構成の方針
- `001_base_v0.2.2.sql`: patch doc が「既にある」前提で参照する object を最小再現
  （alo_law_work / alo_statutes / alo_edges(D2列) / alo_law_succession_edge /
   alo_law_ref_temporal_eval_event / `fn_resolve_law_reference_at` 二段・LIMIT 1 / eval append-only）。
  two-tier CHECK は base に**入れず**、patch が NOT VALID→backfill→VALIDATE で入れる（P0-1 手順の忠実再現）。
- `010_patch_v0.2.3.sql`: patch doc を**逐語転記**（列名のみ実体に合わせた）。
