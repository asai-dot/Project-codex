## 重要な前提
- **Seed（種）であり確定値ではない。** 人手キュレーション＋Salesforce突合が前提（SF本体へは未アクセス）。
- 突合は**依頼者名の正規化による概算照合**。`crosswalk_master.csv` の `box_folder` 列は「現用フォルダ範囲（浅井・武市の受任＋相談）」での一致。
- 列の意味・カバレッジ・漏れ候補・既知の注意点は **`docs/06`** を参照。

## 生成方法（再現）
- 入力: Gmailラベル一覧（API実測）＋ Box案件フォルダ一覧（API実測）。
- 正規化: NFKC・法人格/敬称/括弧書き/空白除去で `client_norm` を作成し突合。
- スクリプトは一時生成物（リポジトリ未収録）。再生成が必要な場合は同手順で再構築。
